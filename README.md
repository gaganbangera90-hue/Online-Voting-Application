# Online Voting System (Flask)

Simple demo online voting site built with Flask, SQLite, and Flask-Login.

Quick start:

1. Create a virtualenv and install deps:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

2. Run the app:

```bash
python app.py
```

3. Open http://127.0.0.1:5000

Default admin: `admin` / `admin` (use only for demo).

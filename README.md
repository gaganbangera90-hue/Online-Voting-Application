# Online Voting Application

## Project Overview

This project is a simple online voting application developed using **Python (Flask)**, **HTML/CSS/JavaScript**, and **SQLite** database. The application allows users to register, log in, and vote for candidates. An administrator can manage candidates and view the results. This project was created as part of a learning exercise to understand web application development with Flask.

## Features

- User registration and login  
- Administrator dashboard to add and manage candidates  
- Voting page for registered users  
- Vote counting and result display  
- Uses SQLite for storing user and vote data

## Technologies Used

- Python  
- Flask framework  
- HTML, CSS, JavaScript  
- SQLite database  
- Flask Login for session management
